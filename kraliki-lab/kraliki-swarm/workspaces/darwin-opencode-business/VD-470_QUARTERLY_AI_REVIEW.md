### Service Delivery Process

### Pre-Review (Week 1)

**Data Collection:**
- Tool usage metrics (from tool dashboards)
- Team surveys (10 min per team member)
- Manager interviews (30 min per department head)
- Budget data (tool subscriptions, costs)

**Automated Data Gathering:**
- API integrations where possible (OpenAI, Anthropic, etc.)
- Export reports from tool dashboards
- Survey forms (Typeform/Google Forms)

---

### Review Week (Week 2)

**Analysis:**
- Tool utilization (metrics, trends)
- ROI calculation (time saved vs cost)
- Adoption assessment (team proficiency)
- Waste identification (unused tools, redundant features)
- Opportunity identification (new tools, workflows)

**Report Drafting:**
- Compile findings
- Create visualizations (charts, graphs)
- Draft recommendations
- Create action plan

---

### Delivery Week (Week 3)

**Deliverables:**
1. **Quarterly AI Review Report** (PDF)
   - Executive summary (1 page)
   - Tool analysis (5-8 pages)
   - ROI breakdown (2-3 pages)
   - Team adoption (2-3 pages)
   - Recommendations (3-5 pages)
   - Action plan (1-2 pages)

2. **Review Call** (30-60 min)
   - Walk through report
   - Discuss findings
   - Answer questions
   - Agree on action items

3. **Implementation Support** (Tier 2 only)
   - Help implement 1-2 key changes
   - Training on new workflows
   - Follow-up after 30 days
