## Action Required

### 1. Enable for TL;DR Bot (@sumarium_bot)
1. Open Telegram and go to **@BotFather**.
2. Type `/mybots` and select **@sumarium_bot**.
3. Go to **Payments**.
4. Select **Telegram Stars**.
5. Enable it.

### 2. Enable for Sense by Kraliki (SenseIt) Bot (@senseit_ai_bot)
1. Open Telegram and go to **@BotFather**.
2. Type `/mybots` and select **@senseit_ai_bot**.
3. Go to **Payments**.
4. Select **Telegram Stars**.
5. Enable it.

## Validation
After enabling, you can try sending `/pay` or `/subscribe` (depending on the bot's commands) to verified it shows the payment modal (even if you don't pay).
