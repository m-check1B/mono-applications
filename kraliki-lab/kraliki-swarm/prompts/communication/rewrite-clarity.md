# Rewrite for Clarity

Transform complex or unclear text into clear, professional communication.

## Prompt Template

```
Rewrite the following text so it is easier to understand. The text will be used in a professional setting. Ensure the tone is clear, respectful, and concise. Text: [paste text].
```

## Parameters

| Parameter | Description | Example |
|-----------|-------------|---------|
| `[paste text]` | The original text to rewrite | Technical jargon, verbose explanation, unclear instructions |

## Example Usage

**Input:**
```
Rewrite the following text so it is easier to understand. The text will be used in a professional setting. Ensure the tone is clear, respectful, and concise. Text: "Pursuant to the aforementioned discussion regarding the implementation of the new procedural methodologies, it has been determined that a comprehensive review of all stakeholder feedback shall be undertaken prior to any decisions being finalized regarding the trajectory of the project moving forward."
```

**Output:**
```
Following our recent discussion, we will review all stakeholder feedback before making final decisions on the project direction.
```

## When to Use

- Simplifying technical documentation for broader audiences
- Editing verbose business writing
- Making instructions clearer
- Preparing summaries for executives
- Translating jargon into plain language
