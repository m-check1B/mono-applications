## Quality Standards

### Document Standards
- All deliverables in client's template (if provided)
- Default: Clean, professional formatting
- Font: [Calibri / Arial / firm default]
- Charts must have clear labels and sources
- Every recommendation needs supporting evidence
- Executive summary required for documents >5 pages

### Analysis Standards
- Data sources must be cited
- Assumptions explicitly stated
- Sensitivity analysis for financial models
- Peer review required before client delivery

### Presentation Standards
- One key message per slide
- No more than 6 bullets per slide
- Visual hierarchy: title → insight → evidence
- Action-oriented recommendations
- Appendix for detailed backup

## Common Tasks

### Client Research
```
1. Industry overview (size, growth, trends)
2. Competitor landscape (key players, positioning)
3. Client context (history, challenges, strengths)
4. Stakeholder map (decision makers, influencers)
```

### Interview Preparation
```
1. Review prior research on interviewee
2. Draft discussion guide (10-15 questions)
3. Prepare hypothesis to test
4. Plan probing follow-ups
5. Create note-taking template
```

### Findings Synthesis
```
1. Aggregate all data sources
2. Identify themes and patterns
3. Prioritize by impact/frequency
4. Develop "so what" for each finding
5. Structure narrative arc
```

### Recommendation Development
```
1. Link each recommendation to a finding
2. Estimate impact (quantify if possible)
3. Assess feasibility (effort, risk)
4. Prioritize (impact vs effort matrix)
5. Define implementation roadmap
```

### Deliverable Creation
```
1. Outline structure (slide titles first)
2. Draft key messages per section
3. Add supporting evidence
4. Design visualizations
5. Peer review → revise
6. Client preview (if appropriate) → final
```

## Engagement Workflow

When working on client engagements:

1. **Engagement setup**
   - Confirm scope, timeline, stakeholders
   - Set up folder structure
   - Kick-off meeting with client

2. **Folder structure**
   ```
   /clients/[client-name]/[engagement-name]/
   ├── 01-admin/           # Contract, scope, logistics
   ├── 02-research/        # External research, data
   ├── 03-interviews/      # Notes, transcripts
   ├── 04-analysis/        # Working documents, models
   ├── 05-deliverables/    # Final outputs
   └── 06-comms/           # Client correspondence
   ```

3. **Weekly rhythm**
   - Monday: Internal alignment, planning
   - Mid-week: Deep work, analysis
   - Friday: Client check-in, progress report
