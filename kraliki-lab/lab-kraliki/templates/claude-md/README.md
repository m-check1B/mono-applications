# CLAUDE.md Template Library

Pre-built CLAUDE.md templates for common Lab by Kraliki Pro customer verticals.

## Available Templates

| Template | Best For | File |
|----------|----------|------|
| **Agency** | Digital agencies, design studios, marketing firms | `agency.md` |
| **Consultancy** | Management, strategy, technology consulting | `consultancy.md` |
| **Legal** | Law firms, legal departments, compliance teams | `legal.md` |
| **Coaching** | Life coaches, executive coaches, trainers | `coaching.md` |

## How to Use

1. Copy the appropriate template to the customer's Lab by Kraliki
2. Work through each section during onboarding
3. Customize terminology, workflows, and standards
4. Remove sections that don't apply
5. Add industry-specific extensions

## Template Structure

Each template includes:

- **Business Context** - What the organization does
- **Terminology** - Industry-specific vocabulary
- **Quality Standards** - Document/code/output requirements
- **Common Tasks** - Typical workflows with step-by-step guides
- **Workflow** - How to organize work and files
- **Forbidden Actions** - Things AI should never do
- **Templates** - Common communication patterns
- **Resources** - Tools and integrations

## Customization Guide

### Quick Start (15 min)
- Fill in [bracketed placeholders]
- Remove irrelevant sections
- Add 3-5 custom terminology entries

### Full Customization (1-2 hours)
- Rewrite business context paragraph
- Add all industry terminology
- Document actual workflows
- Specify exact quality requirements
- Add integration links

## Creating New Templates

When creating templates for new verticals:

1. Start with the consultancy template (most generic)
2. Interview 2-3 practitioners from the industry
3. Identify unique terminology and workflows
4. Note compliance or regulatory considerations
5. Test with a pilot customer
