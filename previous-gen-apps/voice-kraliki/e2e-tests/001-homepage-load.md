# E2E Test 001: Homepage Load

**Priority:** Critical
**Route:** `/`
**URL:** https://voice.verduona.dev

## Objective

Verify the CC-Lite homepage loads correctly and displays essential elements.

## Prerequisites

- None (public page)

## Test Steps

```
1. Navigate to https://voice.verduona.dev
2. Wait for page to fully load (no loading spinners visible)
3. Verify the page title contains "CC-Lite" or "Operator"
4. Check for the presence of:
   - Header/navigation bar
   - Login link or button
   - Main content area
5. Verify no JavaScript errors in console
6. Take a screenshot
```

## Expected Results

- [ ] Page loads within 5 seconds
- [ ] No 404 or 500 errors
- [ ] Navigation elements visible
- [ ] Login/Register links accessible
- [ ] Dark/light theme toggle present (if implemented)
- [ ] No console errors

## Verification Command

```
Navigate to https://voice.verduona.dev
Wait for page to load completely
Take a screenshot
Report what you see on the page
```
