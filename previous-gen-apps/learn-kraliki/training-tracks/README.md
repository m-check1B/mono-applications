## Track Structure

Each industry track includes:

1. **track.json** - Metadata and configuration
2. **overview.md** - Executive summary and business case
3. **modules/** - Detailed module content
4. **assessments/** - Certification requirements
5. **resources/** - Industry-specific tools and templates
