# Dashboard Tour

The Kraliki Swarm dashboard is your central hub for managing all ecosystem tools. Let's explore its main areas.

## The Navigation Bar

At the top of every page, you will find the navigation bar with:

- **Logo** - Click to return to the dashboard home
- **Products** - Quick links to Focus, Speak, Learn, and more
- **Notifications** - Bell icon for alerts and updates
- **Profile** - Your account settings and logout

## Main Dashboard Sections

### 1. Quick Actions

The dashboard home shows your most important actions:

```
+------------------+------------------+
|   Active Tasks   |  Team Feedback   |
|      12 items    |   3 new voices   |
+------------------+------------------+
|   Learning Path  |   AI Agents      |
|   2/5 complete   |   4 active       |
+------------------+------------------+
```

### 2. Product Cards

Each Kraliki product has its own card showing:

- Current status
- Key metrics
- Quick action button

### 3. Activity Feed

The right sidebar shows recent activity across all products:

- Completed tasks
- New feedback received
- Agent actions
- Team updates

## Keyboard Shortcuts

Kraliki supports keyboard navigation for power users:

| Shortcut | Action |
|----------|--------|
| `G` + `D` | Go to Dashboard |
| `G` + `F` | Go to Focus |
| `G` + `S` | Go to Speak |
| `G` + `L` | Go to Learn |
| `/` | Open search |
| `?` | Show all shortcuts |

## Customizing Your Dashboard

You can personalize your dashboard experience:

1. Click **Settings** (gear icon)
2. Select **Dashboard Layout**
3. Drag and drop cards to reorder
4. Toggle visibility of sections

> **Tip:** Pin your most-used product to the top for faster access.
